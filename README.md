# 📊 Student Performance Analysis using EDA & Machine Learning

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) and Machine Learning on the Student Performance dataset to identify the factors affecting students' academic performance and predict final grades.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

Student Performance Dataset

Target Variable:
- G3 (Final Grade)

---

## 📊 Exploratory Data Analysis

The project includes:

- Data Cleaning
- Missing Value Analysis
- Duplicate Value Analysis
- Statistical Summary
- Histograms
- Count Plots
- Pie Charts
- Box Plots
- Scatter Plots
- Pair Plot
- Correlation Heatmap
- Correlation Analysis

---

## 🤖 Machine Learning Models

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

---

## 📈 Model Evaluation

- MAE
- MSE
- RMSE
- R² Score

---

## 📌 Key Findings

- Previous grades (G1 & G2) strongly influence final grades.
- Study time positively impacts performance.
- More failures generally reduce final grades.
- Students with fewer absences perform better.
- Random Forest achieved the best performance.
